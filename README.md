# Job-Hunting-Mail-Automator（就活メール自動作成ツール）

Google Apps Script (GAS) と Gemini API を連携させ、求人票（PDF）から自動でメールアドレスを抽出（OCR）し、応募書類を添付したGmailの下書きをワンクリックで自動生成する就活効率化ツールです。

## 開発の背景と目的
就職活動において、企業ごとの求人票の確認、宛先アドレスの入力、履歴書などの添付ファイルの準備、および定型文の作成には多くの手作業が発生し、誤送信のリスクや時間的コストが課題となっていました。
この課題を解決するため、**「手作業を極限まで減らし、誤送信を防ぐ安全かつ爆速な仕組み」**を自作しました。

## 主な機能
* **AIによる求人票OCR（宛先自動認識）：** 
  Gemini API（`gemini-1.5-flash`）を活用し、求人票PDFの中から担当者のメールアドレス（`@` を含む文字列）を自動で高精度に抽出します。
* **添付ファイルの自動一括集約：** 
  Googleドライブ内の指定フォルダ（「就活」フォルダ）にある履歴書や職務経歴書などの応募書類をすべて自動で検出し、メールに一括添付します。
* **ワンクリック下書き生成：** 
  ハローワーク紹介用の定型文をベースとしたメール本文を自動構築し、Gmailの「下書き」として保存します（いきなり送信されないため、最終確認ができて安全です）。

## システム概要図
[求人票PDF + 履歴書など] ──> Google ドライブ（「就活」フォルダ）
│
▼（実行）
Google Apps Script
│
┌───────────────────────┴───────────────────────┐
▼ (PDFを送信)                                   ▼ (一括添付)
Gemini API                                        Gmailの下書き
(アドレスを自動抽出) ──> [宛先: info@...] ─────────────> [件名・本文・添付完了!]


## 技術スタック
* **言語:** JavaScript (Google Apps Script)
* **AI API:** Gemini API (`gemini-1.5-flash`)
* **Google サービス:** DriveApp, GmailApp, UrlFetchApp

## 導入・設定方法

### 1. フォルダの準備
Googleドライブのルートディレクトリに `就活` という名前のフォルダを作成し、以下のファイルを格納します。
* ターゲットとなる求人票PDF（ファイル名に必ず「求人票」という文字を含めてください）
* 添付したい応募書類（履歴書、職務経歴書など）

### 2. APIキーの取得
[Google AI Studio](https://aistudio.google.com/) から Gemini API キーを発行します。

### 3. GASの配置
1. Googleドライブから「新規」>「その他」>「Google Apps Script」を開きます。
2. 本リポジトリの `Code.js` の内容を貼り付けます。
3. コード1行目の `GEMINI_API_KEY` に、取得したご自身のAPIキーを設定します。
